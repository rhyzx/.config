.config
=======

My self using config/rc for vim/shell/etc..


Install
-------

`$ cd ~`
`$ git clone git@github.com:rhyzx/.config.git & cd .config`


Link rc files
-------------

`$ script/link-rc.sh`


Sync Vim bundle
---------------

`git submodule update --init --recursive`


Scripts
-------

 - **git-submodule-mv.sh** bat move vim-bundle's path
